# Superstore Sales: Executive BI Analysis Dashboard 🏢📊

## 📌 Project Overview
This project is a comprehensive Business Intelligence solution designed to analyze retail performance across North America. Using the Superstore dataset, I developed a multi-page interactive dashboard that transforms raw transactional data into actionable insights for executive decision-making.

## 🛠️ Tech Stack & Skills
- **Power BI Desktop:** Advanced report authoring and multi-page UI/UX design.
- **DAX (Data Analysis Expressions):** Created custom measures for Total Sales ($1.08M), Total Customers (793), and Average Order Value.
- **Data Modeling:** Implementation of a Star Schema to handle large-scale retail datasets.
- **Geospatial Mapping:** Using Map visuals to identify regional sales hubs.

## 🗂️ Dashboard Structure
1. **Executive Overview:** A high-level view of KPIs ($1.08M Total Sales) and category performance.
2. **Regional Analysis:** Interactive map and bar charts pinpointing the West region as the strongest performer.
3. **Customer Analysis:** Deep dive into the "Consumer" vs "Corporate" segments to understand revenue contribution.
4. **Product Analysis:** Identifying "Technology" as the highest revenue-generating category.
5. **Recommendations:** Data-driven strategy suggestions for underperforming regions.

## 🔍 Key Insights
- **Top Performer:** The **Technology** category generates the highest revenue, while the **Consumer** segment contributes the largest share of total sales.
- **Geographic Hubs:** The **West region** shows the strongest sales performance, providing a model for expansion into underperforming territories.
- **Segment Strategy:** Focus on high-value "Corporate" buyers is recommended to increase the Average Order Value (AOV).
